# 查看服务调用明细
平台提供查看服务维度调用API的详细信息。本文介绍如何查看服务调用明细。

## 前提条件
API已被调用。

## 操作步骤
1. 登录 Uniplore Data Studio 平台。
2. 在左侧菜单，单击数据服务菜单，选择服务报表菜单。
3. 在服务调用汇总列表，单击查看明细。

# 查看API操作日志
通过查看API操作日志，了解当前API的使用情况。本文以数据服务API为例介绍如何查看API操作日志。

## 前提条件
已创建数据服务API。

## 操作步骤
1. 登录 Uniplore Data Studio 平台。
2. 在左侧菜单，单击数据服务菜单，选择API开发菜单。
3. 在API开发页面，单击目标API操作列的更多 > 操作日志，查看创建、发布、下线等操作日志。

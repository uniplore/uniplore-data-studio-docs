# 查看API详情
查看API详情有助于用户了解API的基本信息、API配置信息、响应示例、流量控制，便于合理匹配业务。本文以数据服务API为例介绍如何查看API详情。

## 操作步骤
1. 登录 Uniplore Data Studio 平台。
2. 在左侧菜单，单击数据服务菜单，选择API开发菜单。
3. 在API开发页面，单击目标API的名称，查看API详情信息。

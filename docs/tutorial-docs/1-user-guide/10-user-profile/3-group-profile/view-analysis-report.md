# 查看我的分析报告
您可以查看我的分析详情，包括群体分析、群体对比分析分析报告。本文介绍如何查看我的分析报告。

## 前提条件
已保存分析报告。

## 操作步骤
1. 登录 Uniplore Data Studio 平台。
2. 在左侧菜单，单击**画像分析**菜单，选择**我的分析**菜单。
3. 在**我的分析报告**页面，单击分析报告名称。
4. 在分析报告详情页面，即可查看我的分析报告详情。

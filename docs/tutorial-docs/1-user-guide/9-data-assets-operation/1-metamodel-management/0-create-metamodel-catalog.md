# 添加查询目录
您可以通过添加元模型目录，对元模型进行分类管理。本文介绍如何添加元模型目录。

## 操作步骤
1. 登录 Uniplore Data Studio 平台。
2. 在左侧菜单，单击**数据资产运营**菜单，选择**元模型目录管理**菜单。
3. 在**元模型目录管理**页面，单击新增按钮。
5. 在**添加目录**弹窗，输入目录名称、目录编码、上级目录、排序与目录描述，单击**确定**。

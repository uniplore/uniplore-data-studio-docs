# 查看模型命名属性
通过查看模型命名属性，可以查看模型命名属性的基本信息、属性值等。本文介绍如何查看模型命名属性。

## 前提条件
- 已完成逻辑表命名属性创建。

## 操作步骤
1. 登录 Uniplore Data Studio 平台。
2. 在左侧菜单，单击数据建模菜单，选择模型规范菜单，在页面左上角选择模型命名属性页签。
3. 在模型命名属性页面，选择需要查看的命名属性，单击查看。

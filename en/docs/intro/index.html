Data Studio 概述

Data Studio 概述

1. 概述​

Uniplore iDIS 是领先的 AI 驱动的新一代一站式大数据分析产品。 集数据集成(ETL)、多维数据分析、可视化探索分析、数据挖掘与人工智能处理于一体，完全符合理想的大数据分析产品的功能丰富、无需用户编程、智能化程度高、使用简单等特点，帮助用户将 80% 以上的时间真正用于数据分析和创造价值。

2. 功能说明​

2.1 多种数据源无缝接入​

  1. 支持 CSV，Excel 等文本数据格式;
  2. 支持 MySQL，PostgreSQL，Oracle 等主流关系数据库;
  3. 支持 Greenplum 等分布式并行数据库;
  4. 支持 HIVE，HAWQ，KYLIN 等 SQL on Hadoop 大数据系统;
  5. 支持基于多表复杂连接方式集成数据。

2.2 灵活数据集成​

  1. 海量 ETL 组件，内置包含数据抽取、清洗、转换、加载、校验等数据处理组件；
  2. 支持在 Web GUI 中通过可视化拖拽方式构建复杂的ETL任务工作流；
  3. 支持 ETL 任务定时、智能调度。

2.3 字段级数据管理​

  1. 支持字段类型修改，字段隐藏，字段重命名等字段级操作；
  2. 支持基于表达式语言扩展字段计算，内置 30+ 种字段操作函数。

3.4 千万级数据秒级响应​

  1. 内置优秀的分布式存储引擎，提高数据响应速度。在 MOLAP 模式下，对于数十TB级的数据规模，在 8 节点的（推荐配置的）集群规模环境中，95% 的多维查询可在 5 秒内返回结果；
  2. 支持分布式内存计算，In-Database 库内计算。

2.5 多种数据分析方式​

可视化分析​

  1. 支持 20+ 种常规图表类型(含矢量地图和离/在线百度地图)；
  2. 支持在 Web GUI 中可视化拖拽定义报表和仪表盘分析任务；
  3. 支持图表可视化动态联动和图表的多维多层级下钻分析；
  4. 支持仪表盘灵活交互，可通过“鼠标点击+菜单选择”的向导式方式来设置复杂交互和联动操作。

多维分析​

  1. 多种分析执行引擎
    支持在 Web GUI 中以可视化拖拽方式定义可并行执行的大数据多维 ROLAP 和 MOLAP 任务，支持以SQL on Hadoop 大数据系统作为 ROLAP 和 MOLAP 多维分析的执行引擎；
  2. 灵活快速多维分析
    支持在 Web GUI 中定义和执行不限层级数的 Drilldown (下钻)，Roll up (上卷)、过滤等 多维分析操作；
  3. 多种多维分析类型和语言
    支持以 SQL 和 MDX 语言作为分析语言；
  4. 可扩展度量字段
    通过简单鼠标和键盘即可建立新的度量字段；
  5. 模型可视化构建
    支持以 Web GUI 方式构建多维模型；
  6. 复杂多维操作
    支持多表复杂连接，分组过滤。

智能分析（数据挖掘）​

  1. 海量算法支持
    内置多种数据挖掘算法、数据预处理及模型评估组件；
  2. 灵活快速建模
    支持基于拖拽式可视化工作流方式构建数据挖掘等复杂分析应用；
  3. 高级 AI 分析
    支持基于 AutoML（自动机器学习）技术的高级AI数据分析技术，用户无需选择特征属性、算法模型和参数即可进行数据挖掘；
  4. 良好的可扩展
    支持引入 Web 服务添加新的智能分析组件实现扩展。

2.6 数据及资源共享​

  1. 不同用户建立的分析资源（多维报表，仪表盘，智能分析工作流）可直接分享给指定用户；
  2. 可通过授权方式将数据资源（数据源，数据集）分享给不同用户。

2.7 集成和开放​

  1. 多维报表分析、可视化分析、智能分析的可视化图表（仪表盘） 可无缝嵌入、集成到第三方 Web 应用中；
  2. 多维分析、可视化分析、智能分析的结果数据，可通过提供的接口，被第三方 Web 和桌面软件应用系统直接获取和使用。

2.8 多种部署模式​

  1. 提供公有云 SaaS 应用（基于 Amazon AWS 中国）；
  2. 支持企业私有部署；
  3. 支持企业在私有数据中心中以 SaaS 模式部署。

3. 产品优势​

3.1 基于大数据技术的优异性能​

Uniplore iDIS 是一款真正支持海量数据的大数据分析智能服务产品。在实际实施的项目案例中，Uniplore iDIS 对于分组汇总表的计算，亿级数据规模，多维分析计算可在秒级时间内完成响应处理。

3.2 开放架构方便集成​

Uniplore iDIS 可结合各类数据存储/处理系统使用，也可以直接连接 ERP/CRM 等业务系统的数据源，扩展现有系统的功能，更好的利用现有 IT 资产。Uniplore iDIS 采用开放的架构，能够方便地融合进其他应用系统，能方便实现与第三方系统的用户和权限整合。

3.3 易于非技术人员使用​

传统 BI 的繁杂之处主要体现在两个方面：技术人员需要花费大量时间准备数据，业务人员缺乏专业技术技能，基于数据来自助实现分析需求的过程复杂。 Uniplore iDIS 提供完整的基于可视化拖拽的从数据集成到多维分析、可视化分析展现等全套数据分析功能支撑，协助用户自助进行数据准备和数据分析。由于上述整个过程中无需用户进行任何编程，大大降低了非 IT 人员使用产品的门槛。 

3.4 实施周期短，维护简单​

由于 Uniplore iDIS 特有的拖拽式数据准备和工作流式可视化数据集成功能，除去原始需求沟通时间，技术人员在实施时无需花费过多的时间来进行数据处理。Uniplore iDIS 特有的建模流程能够极大的减少需求多变导致的数据准备和沟通交流时间，真正做到了实施周期短，后期也无需投入过多的人力和时间来维护系统，较好的解决了需求的变动带来的种种问题。
